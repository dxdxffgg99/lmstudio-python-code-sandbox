# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-01-30

### Added
- Initial release of Python Code Sandbox plugin for LMStudio
- `run_python` tool for executing Python code snippets
- `install_package` tool for pip package management (install/uninstall/list)
- Virtual environment (.venv) setup scripts for Windows and Unix
- Cross-platform pip wrapper for npm integration
- Full TypeScript rewrite with strict typing
- Comprehensive error handling and cleanup
- `.gitignore` for common build artifacts
- LICENSE (MIT) and CONTRIBUTING.md documentation
- README with complete usage guide

### Features
- Execute Python code without timeout limits
- Automatic .venv prioritization for AI package usage
- Error handling with detailed stderr output
- Cleanup of temporary scripts
- Support for Windows (PowerShell) and Unix (sh) environments
